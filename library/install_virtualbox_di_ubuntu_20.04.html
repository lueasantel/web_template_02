Install Virtualbox di ubuntu 20.04

Publikasi pada 7 October 2021

Downloads Files

1. Download virtualbox di Homepage Virtualbox sesuaikan dengan versi yang digunakan.

2. Download juga VirtualBox Extension Pack ini disarankan untuk menambakan beberapa fitur pada virtualbox

Cara install

1. Buka terminal pada ubuntu melalui menu aplikasi atau biasanya menggunakan shortcut CTRL + ALT + T lalu Enter

2. Ikuti perintah dibawah ini:

~$ cd Downloads ==> sesuaikan dengan lokasi penyimpanan file virtualbox yang di downlaod. Di sini saya menyimpannya di folder Downloads.

~$ sudo dpkg -i virtualbox-*.deb ==> nama "virtualbox.deb" disesuaikan dengan nama file yang didownloads.

3. Tunggu hingga selesai proses installasinya dan jangan lupa lihat pada bagian yang di install jika ada coment yang seperti ini : ~$ dpkg: error processing package virtualbox-* (--install) atau beberapa paket yang :however:

4. Gunakan perintah berikut untuk memperbaikinya :

~$ sudo apt-get -f install ==> perintah ini berfungsi untuk menginstall paket yang belun terinstall atau tertinggal pada saat proses installasi sebelumnya.

Pemasangan VirtualBox Extention Park

1. Buka virtualbox yang telah diinstall.

2. Buka menu File -> Preferences (CTRL+G).

3. Pilih menu Extensions.

4. Klik tombol + untuk instalasi Extension Package, buka file extension yang telah didownload sebelumnya. Masukkan password untuk melanjutkan instalasi.

5. Tunggu hingga selesai dan virtualbox siap digunakan.
