After Install Ubuntu 20.04 Focal Fossa

Publikasi pada 7 October 2021

1. Check For Updates

Lakukan update repository pada ubuntu untuk medapatkan paket terbaru. melalui terminal maupun dengan gui

~$ sudo apt update && sudo apt upgrade -y

2. Enable Partner Repositories

Aktifkan repository mitra untuk mendapatkan tambahan dukungan driver melalui menu "Software & Update"

3. Install Missing Graphic Driver

Buka "Software & Update" lanjut ke menu "Additional Drivers tab."

Di bawah tab Additional Drivers, Anda dapat memilih apakah akan menginstal perangkat lunak berpemilik sumber tertutup atau open source x.org Xerver Driver.

Menginstal Proprietary Drivers berpemilik adalah yang direkomendasikan dalam daftar.

4. Installing Complete Multimedia Support

Untuk install third-party software for graphics and Wi-Fi hardware, Flash, MP3, and other media

~$ sudo apt install ubuntu-restricted-extras

5. Install Microsoft Fonts

Jika memerlukan font dari Microsoft untuk kebutuhan membuat dokumen

6. Install Popular and Most useful Ubuntu software

Lakukan instalasi pada software yang memudahkan dalam melakukan produktivitas

~$ sudo apt install gimp vlc dll.

7. Install GNOME Shell Extensions

Untuk mengubah tampilan ubuntu dengan costumisasi sesuai keinginan hompagenya di Gnome Extensions

8. Disable Startup Applications

Untuk mempercepat proses memuat di ubuntu. bisa di nonaktifkan pada menu "Startup Application".

9. Install Preload

Untuk mempercepat loading aplikasi dengan mengutamakan penggunaan ram terlebih dahulu.

~$ sudo apt install prelood
