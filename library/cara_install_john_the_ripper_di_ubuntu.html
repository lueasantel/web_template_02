Cara Install John The Ripper di Ubuntu

Publikasi pada 15 October 2021

Persiapan

1. Ubuntu harus sudah terinstall dengan git.

2. Cara installnya dengan menggunakan perintah pada terminal :

~$ sudo apt install git

Cara Install

1. Kunjungi Github openwall/john.

2. Clone scripnya dengan perintah :

~$ git clone https://github.com/magnumripper/JhonTheRipper.git

3. Kemudian masuk ke direktori "src" pada file johntheripper

~$ cd ./JhonTheRipper/src

4. Kemudian update ubuntu dengan perintah :

~$ sudo apt-get update

5. Lalu install pakage pendukung software john the ripper dengan perintah :

~$ sudo apt-get install libssl-dev

Lalu lakukan installasi dengan perintah :

~$ ./configure && make

Tunggu hingga proses selesai dan John the ripper siap digunakan.

Note : untuk menjalankannya melalui direktori "~$ cd JhonTheRipper/run"
