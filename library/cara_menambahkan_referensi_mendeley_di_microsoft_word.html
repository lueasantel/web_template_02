Cara Menambahkan Referensi Mendeley di Microsoft Word

Publikasi pada 10 October 2021

Download Files

1. Download file Mendeley dekstop pada halaman Homepage Mendeley

2. Kemudiaan install Mendeley pada PC atau Laptop

Cara Memghubungkan Mendeley ke Microsoft Word

1. Sebelumnya daftar terlebih dahulu di website Mendeley dengan email

2. Buka aplikasi mendeley

3. Kemudian ke arah menu "tools", Kemudian pilih "install MS Word plugin"

4. Kemudian cek pada aplikasi Microsoft Word apakah sudah terinstal pada bagian menu "referensi" jika berhasil di install akan ada menu mendeley

Cara menggunakan Mendeley di Microsoft Word

1. cari referensi dalam bentuk *.pdf

2. Buka aplikasi Mendeley

3. Buka menu "file" kemudian pilih "add file"

4. Kemudian pilih file referensi yang sudah di cari (dalam format *.pdf)

5. Jika berhasil akan ada file referensi yang ditambahkan muncul di aplikasi Mendeley dan akan ada keterangan tentang judul, actor, tahun, dan lainnya mengenai file referensi yang ditambahkan

6. kemudian pilih menu "syncron" untuk membiarkan aplikasi mandeley membaca file referensi yang di tambahkan

7. Jika sudah, kemudian buka aplikasi Microsoft Word buka menu "referensi"

8. Arahkan cursor ke bagian yang akan dikutip dengan mendeley

9. Kemudian pilih menu "incert citation"

10. Kemudian akan muncul pop up mendeley, lalu pada kotak pencarian cari judul referensi yang telah terbaca di Mendeley, kemudian pilih "OK"

11. Maka pada bagian tempat cursor akan ditambahkan kutipan berupa referensi dan untuk mengubah style Mendeley pilih "style Mendeley" untuk style default akan berkutip [nama actor, tahun terbit]

12. Dan untuk memasukkan referensi ke dalam daftar pustaka

13. Arahkan kembali cursor pada bagian daftar pustaka

14. Lalu pilih menu “Insert Bibliography”

15. Maka referensi akan otomatis ditambahkan di bagian dafta pustaka

16. Demikian cara menggunakan Mendeley pada Microsoft Word
