Install Microsoft Office 2007 (32bit) di Ubuntu 20.04

Publikasi pada 8 October 2021

Sebelum Install Microsoft Office

Note : semakin lama versi Microsoft Office semakin baik dan stabil panggunaannya (32bit)

1. Buka terminal atau menggunakan shortcut CTRL + ALT + T pada keyboard.

2. Lakukan update repository

~$ sudo apt-get update

3. Lanjut install play on linux

~$ sudo apt-get install playonlinux

4. kemudian install winbind, karena Microsoft Office membutuhkan winbox

~$ sudo apt-get install winbind

Install Microsoft Office

1. Buka Play On Linux

2. Lanjut ke menu "Office"

3. Scroll dan cari Microsoft Office (sesuaikan dengan file Microsoft Office yang akan diinstall)

4. tekan "install", lanjut tekan "browse" dan pilih file Microsoft Office yang akan di install

5. tekan "next" dan kemudian akan ada konfirmasi tentang instalasi wine mono installer kemudian klik "install"

6. Dan jika ada konfirmasi instalasi "wine gecko" kemudian klik "install"

7. Dan jika muncul kinstalasi "windowsxp" sesuai keinginan (dicancel juga tidak berpengaruh pada Microsoft Office yang diinstall)

8. Lanjut ke instalasi Microsoft Office seperti di windows
