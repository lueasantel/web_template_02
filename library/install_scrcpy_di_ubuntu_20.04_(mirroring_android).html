Install Scrcpy di Ubuntu 20.04 (Mirroring Android)

Publikasi pada 8 October 2021

Sebelum Intall

1. Sebelum menginstall scrcpy sebaiknya kita menginstall dulu ADB (Android Debug Bridge), aplikasi ini berguna untuk mengatur perangkat Android dari PC. Lakukan perintah ini di Terminal.

~$ sudo apt install android-tools-adb android-tools-fastboot

Cara install

Setelah ADB terinstall baru kita install scrcpy, untuk Ubuntu Linux versi 20.04 ke atas, aplikasi ini sudah tersedia di repo Ubuntu, untuk Ubuntu Linux versi dibawahnya kita perlu menggunakan installer snapd

~$ snap install scrcpy

Cara Penggunaan

Melalui Kabel USB

1. Sebelum menggunakan bisa menggunakan scrcpy, kita perlu mengatur terlebih dahulu perangkat Android kita. Kita perlu mengaktifkan USB debugging.

2. Caranya masuk dulu ke Developer Option dan aktifkan USB Debugging

3. Jika Ingin PC bisa mengoperasikan perangkat Android lewat mouse dan keyboard di PC, aktifkan juga USB Debugging (Security Settings)

4. Sambungkan peranggkat Android ke PC lewat kabel USB, Beri izin permission Allow USB Debugging pada perangkat Android. Jalankan perintah ini diterminal untuk mencek apakah PC sudah terhubung dengan perangkat Android

~$ adb devices ==> Jika sudah tersambung akan muncul kode perangkat dibawah "List of devices attached"

5. jalankan perintah untuk memulai scrcpy

~$ scrcpy

Melalui WIFI (Not Recommended ==> lagging)

Note : PC dan perangkat Android harus terhubung di jaringan yang sama

1. pertama kita harus memberi akses port 5555 pada perangkat Android kita. Untuk itu jalankan perintah ini di terminal (PC dan perangkat Android masih tersambung lewat USB)

~$ adb tcpip 5555

2. Kemudian cari IP yang digunakan perangkat Android kita, caranya mudah tinggal lihat dari "About Phone", Cari "Status"

3. Dibagian "Ip Address" contohnya [192.1168.6.265]

3. Jalankan perintah ini untuk memulai scrcpy

~$ adb connect 192.1168.6.265:5555 ==> "192.1168.6.265" sesuaikan ip address dengan yang ada di android

Koneksi Lebih Dari Satu Devices

1. Gunakan perintah adb untuk melihat nama device yang terhubung

~$ adb devices ==> contohnya : 192.168.6.265:5555 [device]

2. Kemudian gunaakan perintah ini untuk mengakses setiap device yang terhubung

~$ scrcpy -s 192.168.6.265:5555 ==> "192.168.6.265:5555" sesuaikan dengan device yang terhubung

3. Jika terjadi lagging (melambat), karena masalah koneksi kabel atau wirelessnya, coba kurangi resolusi atau video bit rate nya

~$ scrcpy -b2M -m800
