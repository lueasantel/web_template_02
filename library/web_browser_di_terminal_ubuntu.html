Web Browser di Terminal Ubuntu

Publikasi pada 15 October 2021

1. W3M

W3m adalah peramban web berbasis teks sumber terbuka yang populer untuk terminal. Meskipun proyek aslinya tidak lagi aktif, versi aktifnya dikelola oleh pengembang yang berbeda, Tatsuya Kinoshita. w3m cukup sederhana, mendukung koneksi SSL, warna, dan gambar sebaris juga. Tentu saja, tergantung pada sumber daya apa yang Anda coba akses, hal-hal mungkin terlihat berbeda di pihak Anda. Sesuai tes cepat saya, sepertinya tidak memuat DuckDuckGo tetapi saya bisa menggunakan Google di terminal dengan baiK. Anda cukup mengetik w3m di terminal untuk mendapatkan bantuan setelah instalasi. Jika Anda penasaran, Anda juga dapat melihat repositori di GitHub .

W3M tersedia di sebagian besar repositori default untuk semua distribusi Linux berbasis Debian. Jika Anda memiliki distro berbasis Arch, Anda mungkin ingin memeriksa AUR jika tidak tersedia secara langsung.

Cara Install W3M di Ubuntu

Menggunakan perintah berikut :

~$ sudo apt install w3m w3m-img

Untuk memulai, Anda cukup mengikuti perintah di bawah ini :

~$ w3m google.com

Anda perlu mengganti google.com ke situs web mana pun yang ingin Anda telusuri. Terakhir, Anda harus tahu bahwa Anda dapat menggunakan tombol panah keyboard untuk menavigasi dan tekan enter saat Anda ingin mengambil tindakan

Untuk keluar, Anda dapat menekan SHIFT+Q

Untuk kembali ke halaman sebelumnya — SHIFT+B

Pintasan tambahan termasuk SHIFT + T untuk membuka tab baru dan SHIFT + U untuk membuka URL baru

2. LYNX

Lynx adalah browser baris perintah open source lain yang dapat Anda coba. Untungnya, lebih banyak situs web cenderung berfungsi saat menggunakan Lynx, jadi saya akan mengatakan itu pasti lebih baik dalam aspek itu. Saya dapat memuat DuckDuckGo dan membuatnya berfungsi. Selain itu, saya juga memperhatikan bahwa ini memungkinkan Anda menerima/menolak cookie saat mengunjungi berbagai sumber daya web. Anda dapat mengaturnya untuk selalu menerima atau menolak juga. Jadi, itu hal yang baik. Di sisi lain, ukuran jendela tidak berubah dengan baik saat menggunakannya dari terminal. Saya belum mencari solusi untuk itu, jadi jika Anda mencoba ini, Anda mungkin ingin melakukannya. Dalam kedua kasus, ini berfungsi dengan baik dan Anda mendapatkan semua instruksi untuk pintasan keyboard tepat saat Anda meluncurkannya di terminal.

Cara Install LYNX di Ubuntu

Untuk Ubuntu, Anda hanya perlu mengetikkan :

~$ sudo apt install lynx

Untuk memulai, Anda hanya perlu mengikuti perintah di bawah ini :

~$ lynx google.com

Anda hanya perlu mengganti situs web contoh dengan sumber daya yang ingin Anda kunjungi

3. LINKS2

Links2 adalah peramban berbasis teks yang menarik yang dapat Anda gunakan dengan mudah di terminal Anda dengan pengalaman pengguna yang baik. Ini memberi Anda antarmuka yang bagus untuk mengetikkan URL dan kemudian melanjutkan segera setelah Anda meluncurkannya

Perlu dicatat bahwa tema akan tergantung pada pengaturan terminal Anda, saya menetapkannya sebagai "hitam-hijau", maka inilah yang Anda lihat. Setelah Anda meluncurkannya sebagai browser baris perintah, Anda hanya perlu menekan tombol apa saja untuk memunculkan URL prompt atau Q untuk keluar darinya. Ini berfungsi cukup baik dan membuat teks dari sebagian besar situs. Tidak seperti Lynx, Anda tidak mendapatkan kemampuan untuk menerima/menolak cookie. Selain itu, tampaknya berfungsi dengan baik.

Cara Install LINKS2 di Ubuntu

Anda dapat menginstalnya dengan mengetikkan perintah berikut di terminal :

~$ sudo apt install links2

4. ELINKS

ELinks mirip dengan Links2 — tetapi tidak lagi dipertahankan. Anda masih akan menemukannya di repositori default dari berbagai distribusi, oleh karena itu, saya menyimpannya di daftar ini. Itu tidak berbaur dengan tema terminal sistem Anda. Jadi, ini mungkin bukan pengalaman yang bagus sebagai browser berbasis teks tanpa mode "gelap" jika Anda membutuhkannya

Cara Install ELINKS di Ubuntu

Anda hanya perlu mengetikkan yang berikut di terminal :

~$ sudo apt install elinks
