Install VirtualBox di Windows

Publikasi pada 7 October 2021

Downloads Files

1. Download virtualbox di Homepage Virtualbox Pilih Windows host pada bagian VirtualBox x.x.x platform packages

2. Download juga VirtualBox Extension Pack ini disarankan untuk menambakan beberapa fitur pada virtualbox

Cara install

1. Klik 2 kali pada file file virtualbox yang telah di downloads.

2. Pada jendela instalasi klik "Next".

3. Pada menu costum setup terdapat beberapa fitur diantaranya :

VirtualBox USB Support : fitur agar guest OS dapat mengenali USB yang menancap di komputer

VirtualBox Networking : fitur agar guest OS dapat terhubung ke internet/jaringan

VirtualBox Phyton 2.x Support : dukungan untuk script Phyton Kami menyarankan agar Anda membiarkan pengaturan tersebut apa adanya

4. Kemudian pastikan lokasi instalasi sudah benar dan disk usage yang memiliki memori yang cukup.

5. Jika muncul jendela Warning: Network Interface yang memberitahukan bahwa koneksi jaringan akan ter-reset dan terputus sementara waktu saat instalasi. Anda tidak perlu kuatir karena hal ini hanya seperti mematikan dan menyalakan internet, bukan mereset setting internet. Kemudian klik "Yes"

6. Kemudian klik "install"

7. Pada Windows 7 dan 8 mungkin akan muncul jendela User Account Control yang meminta konfirmasi untuk menginstall VirtualBox. Kemudian klik "Yes"

8. Saat tengah proses install, akan muncul kotak dialog Windows Security. Beri tanda centang pada Always trust software from “Oracle Corporation”. Hal ini untuk mencegah Windows memunculkan kotak dialog serupa ketika akan menginstall aplikasi lainnya dari Oracle. Kemudian klik "Install"

9. Tunggu hingga proses instalasi selesai

Pemasangan VirtualBox Extention Park

1. Buka virtualbox yang telah diinstall.

2. Buka menu File -> Preferences.

3. Pada menu Extensions, Klik tombol + untuk instalasi Extension Package, buka file extension yang telah didownload.

4. Akan muncul kotak dialog konfirmasi install dan keterangan tentang versi extension pack. Kemudia klik "Install"

5. Selanjutnya kotak dialog lisensi VirtualBox akan muncul. Setelah selesai membaca, scroll ke bawah kemudian klik I Agree ( Anda harus scroll ke bawah agar tombol I Agree dapat ditekan )

6. Jika ada jendela informasi instalasi. Kemudia klik "OK"

7. kemudian virtualbox siap digunakan.
