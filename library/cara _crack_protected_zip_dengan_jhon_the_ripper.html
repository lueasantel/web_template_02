Cara Crack Protected Zip dengan John The Ripper

Publikasi pada 15 October 2021

Persiapan

1. Install terlebih dahulu john the ripper.

2. pada tautan berikut terdapat tutorial cara install john the ripper

Cara Penggunaan John The Ripper

1. Siapkan file "contoh.zip" yang akan di crack.

2. Lalu simpan di direktori /JohnTheRipper/run

3. Kemudian Masuk ke direktori "run" dengan terminal dengan perintah :

~$ cd JohnTheRipper/run

4. Kemudian ketik perintah :

~$ ./zip2john contoh.zip > zip.hash ==> "contoh.zip sesuaikan dengan nama file zip yang akan di crack

5. Kemudian cek apakah sudah terbuat file dengan nama "zip.hash" dengan perintah "ls"

6. Jika ada, Kemudian ketik perintah berikut untuk memulai crack :

~$ ./john zip.hash

7. Jika berhasil akan tampil password yang di crack tadi.

Note : anda dapat memodifikasi kumpulan password pada bagian password.lst

1. Buka terminal lalu arahkan ke direktori "run"

~$ cd JohnTheRipper/run

2. Lalu cari file password.lst jika ada ketik perintah berikut untuk memodifikasinya.

~$ sudo nano password.lst

3. ketik shortcut CTRL + X lalu "yes" untuk menyimpan perubahan
