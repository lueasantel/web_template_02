Install Google Chrome di ubuntu 20.04

Publikasi pada 7 October 2021

Downloads Files

Melalui Web Browser

1. Buka web browser default yang ada pada ubuntu contohnya seperti : firefox.

2. Kemudian ke halaman homepage Google Chrome kemudian di downloads.

3. Klik "download chrome" lalu pilih file yang berformat ".deb" dan pilih 32bit / 64bit sesuai spesifikasi laptop / komputer.

5. kemudian download.

Melalui Terminal

1. Buka terminal lalu menuju ke folder ~/Downloads dengan perintah

~$ cd Downloads ==> sesuai dengan keinginan masing - masing mau di folder apapun.

~$ wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb ==> jika belum ada wget bisa di install dulu dengan perintah [~$ sudo apt install wget] pada terminal.

2.cek dengan perintah "ls". Di sana sekarang terdapat file dengan nama "google-chrome-stable_current_amd64.deb".

Cara Install

1. Buka terminal

~$ cd Downloads ==> tempat file google-chrome.deb.

2. lalu gunakan peintah berikut :

~$ sudo dpkg -i google-chrome-stable_current_amd64.deb ==> sesuaikan nama "google-chrome-stable_current_amd64.deb" dengan nama yang dipakai.

3. Tunggu hingga selesai proses installasinya dan jangan lupa lihat pada bagian yang di install jika ada coment yang seperti ini :

~$ dpkg: error processing package virtualbox-* (--install) atau beberapa paket yang : however:

4. Gunakan perintah berikut untuk memperbaikinya :

~$ sudo apt-get -f install ==> perintah ini berfungsi untuk menginstall paket yang belun terinstall atau tertinggal pada saat proses instalasi sebelumnya.

5. Kemudian Google Chrome siap digunakan
